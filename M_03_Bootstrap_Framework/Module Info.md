## Module Overview

### Lesson 1: Introduction to Bootstrap
- **Objective:** Understand what Bootstrap is and how to include it in a project.
- **Key Concepts:** CDN links, basic template, responsive design.

### Lesson 2: Typography
- **Objective:** Learn how to use Bootstrap's typography classes to style text.
- **Key Classes:** Headings, display headings, lead, text alignment.

### Lesson 3: Containers in Bootstrap
- **Objective:** Understand the container system and the grid layout.
- **Key Classes:** `.container`, `.container-fluid`, rows, and columns.

### Lesson 4: Bootstrap utilities part 1
- **Objective:** Explore common utility classes for spacing and colors.
- **Key Classes:** Margin, padding, text colors, background colors.

### Lesson 5: Bootstrap utilities part 2
- **Objective:** Learn about borders, shadows, and display utilities.
- **Key Classes:** Borders, rounded corners, shadows, visibility.

### Lesson 6: Bootstrap Carousel and Scrollspy
- **Objective:** Implement interactive components like image sliders and navigation tracking.
- **Key Components:** Carousel, Scrollspy.
