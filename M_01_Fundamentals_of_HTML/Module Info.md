## Module Overview

### Lesson 1: Setting up Dev Tools & Intro To HTML
- **Objective:** Understand the basic structure of an HTML document (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
- **Key Tags:** `<html>`, `<head>`, `<title>`, `<body>`, `<br>`.

### Lesson 2: Structure Elements
- **Objective:** Learn how to use headings, paragraphs, and text formatting tags.
- **Key Tags:** `<h1>` to `<h6>`, `<p>`, `<b>`, `<strong>`, `<i>`, `<em>`, `<mark>`, `<small>`, `<del>`, `<sup>`, `<sub>`.

### Lesson 3: Links, Tables and Lists
- **Objective:** Create navigation using hyperlinks, organize data with tables, and group items using ordered and unordered lists.
- **Key Tags:** `<a>`, `<ul>`, `<ol>`, `<li>`, `<table>`, `<tr>`, `<th>`, `<td>`.

### Lesson 4: Media in HTML
- **Objective:** Enhance web pages by embedding images, audio, video, and external content using iframes.
- **Key Tags:** `<img>`, `<audio>`, `<video>`, `<source>`, `<iframe>`.

### Lesson 5: Semantic Elements in HTML
- **Objective:** Use semantic tags to provide meaning to the web page structure, improving accessibility and SEO.
- **Key Tags:** `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<figure>`, `<figcaption>`, `<footer>`.

### Lesson 6: Forms in HTML
- **Objective:** Build interactive forms to collect user input using various input types, labels, and grouping elements.
- **Key Tags:** `<form>`, `<label>`, `<input>` (text, password, radio, checkbox, date, number, submit), `<fieldset>`, `<legend>`, `<select>`, `<option>`, `<output>`.
