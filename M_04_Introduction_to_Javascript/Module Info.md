## Module Overview

### Lesson 1: Introduction to Javascript
- **Objective:** Understand the role of JavaScript in web development and learn basic syntax.
- **Key Concepts:** Variables, data types, basic operators, console.log.

### Lesson 2: Introduction to Functions in Javascript
- **Objective:** Learn how to define and call functions to reuse code.
- **Key Concepts:** Function declaration, parameters, return values, arrow functions.

### Lesson 3: Introduction to Methods, Loops and Switches in JavaScript
- **Objective:** Master control flow using loops and switch statements, and explore built-in methods.
- **Key Concepts:** For loops, while loops, switch statements, string and number methods.

### Lesson 4: Introduction to Objects in JavaScript
- **Objective:** Understand how to group related data and functionality using objects.
- **Key Concepts:** Object literals, properties, methods, 'this' keyword.

### Lesson 5: Introduction to Array and call stack
- **Objective:** Learn how to store and manipulate lists of data using arrays and understand the execution context.
- **Key Concepts:** Array methods (push, pop, shift, unshift), indexing, call stack, recursion.

### Lesson 6: Introduction to HTML DOM
- **Objective:** Learn how to interact with and modify HTML elements using JavaScript.
- **Key Concepts:** Document object, selecting elements, event listeners, changing styles and content.
